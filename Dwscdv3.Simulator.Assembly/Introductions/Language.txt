语言

1. 关键字 - 未完成
	关键字类似于汇编语言，关键字清单详见 Language 类的 Command 成员。

2. 标签 - 未开始
	标签为一串字母后紧跟一个 半角冒号 : 。
	可使用 jmp 标签名 使程序流程跳转到该标签名后的语句。

3. 注释 - 未开始
	注释类似于C语言，使用 // 或 /* ... */
	// - 单行注释
		直接读取下一行
	/* ... */ - 多行注释
		检测到 /* 开头且不位于字符串中时，忽略所有内容直到遇到 */ 结尾

4. 运算符 - 未开始
	运算符列表
		&		&用于获取当前执行代码所在行数。
				一个典型的用法是使用 jmp 跳转前记录当前代码所在行数以便跳转回来。（就像执行一个子程序一样）
		[ ]		方括号 [ ] 用于访问内存。
		{ }		花括号 { } 用于访问视频内存。

5. 运算和比较 - 未开始
	运算和比较类似于汇编语言，使用指令而不使用运算符。
	指令列表
		运算：
			add x, y	加
			sub x, y	减
			mul x, y	乘
			div x, y	除
			mod x, y	取余
			mov x, y	赋值
		比较：(如果结果为真，则执行下一条语句，否则跳过下一条语句
			eql	x, y	等于
			grt x, y	大于
			les x, y	小于

6. 字符串 - 未开始
	字符串使用 半角双引号 " 括起，使用 反斜杠 \ 指定转义字符
